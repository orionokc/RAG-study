# Smart Notes
Smart Notes is an AI-powered study stool. After uploading their notes (pdf file), users can choose to have AI automatically generate flash cards from their notes and/or consult and ask questions to their notes, as Smart Notes trains existing LLMs on the user-upload notes via RAG (Retrieval-Augmented Generation). Users also have the option to define how many flash cards to develop and the difficulty of the flash cards, as well. 
Unfortunately, Smart Notes is not deployed yet, but users can run the app by cloning the code in the app.py file in this repository. After cloning the code, the app can be launched by navigating to the directory that contains the app.py file in terminal and then running the command streamlit run app.py
